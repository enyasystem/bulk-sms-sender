BELFX Fintech Platform

Product Requirements Document (PRD) & API Overview

1. Project Overview

BELFX is a Nigerian-focused fintech platform that enables peer-to-peer (P2P) currency transactions between users — primarily for NGN ↔ USD, CAD, EUR, and GBP. The system earns through commissions on every successful transaction. The MVP focuses on wallet management, user authentication, offer matching, payment integration, and KYC.

2. Core Features

  • Multi-currency wallet system (NGN, USD, GBP, EUR, CAD)
  • User registration and login (via Supabase Auth)
  • BVN and ID-based KYC verification flow
  • P2P offer creation with rate control
  • Transaction logging and real-time status updates
  • Escrow simulation and commission deduction
  • Admin panel to verify KYC and monitor activity

3. API Platforms & Integrations

API Platform Purpose Cost Estimate
Supabase Auth, PostgreSQL DB, Storage, Edge Functions Free / N37,000+ monthly (Pro)
Flutterwave NGN deposits, withdrawals, and virtual accounts ~1.4% per transaction
IdentityPass BVN, NIN, ID, Selfie KYC N150 – N250 per user
ExchangeRate-API FX data (NGN/USD, GBP, CAD, EUR) Free / N23,000+ (Pro)
Termii (Optional) OTP, 2FA, and SMS notifications N2 – N5 per SMS

4. MVP Tech Stack Summary

  • Frontend: v0.dev + Tailwind CSS
  • Backend: Supabase (PostgreSQL, Auth, Storage)
  • Payments: Flutterwave
  • KYC: IdentityPass
  • Rates: ExchangeRate-API
  • Optional: Termii for SMS & OTP

5. Estimated MVP Development Costs

  • Supabase: Free or Pro (N37,000+/mo)
  • KYC for 100 users: N15,000 – N25,000
  • Flutterwave test volume: ~N10,000+
  • Domain & Hosting: N15,000 – N25,000
  • Termii credits: N5,000 – N10,000 (Optional)

6. Benefits of Free-Tier APIs

  • Reduce MVP development costs significantly
  • No monthly fees until scale is needed
  • Secure, production-ready tools out of the box
  • Scalable infrastructure with auto-upgrade options
